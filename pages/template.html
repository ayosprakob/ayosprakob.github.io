Installation
Download the files from here. Or use git clone in the terminal: 

    git clone https://github.com/ayosprakob/ayosprakob.github.io
Page components
The webpage has several components that can be edited manually.

$\odot$ Viewport

This is the main column that displays everything. You can adjust the style of the viewport in style.css

$\odot$ Header

This is located at the top of the viewport. You can edit its content in the function updatepage() in script.js

$\odot$ Nevigation panels

Usually this shows the link to related pages. You can modify this at the end of each html file. It's the argument of the function updatepage(). If you want to remove the navigation panels, just delete the tag <div id="nav1"></div> and <div id="nav2"></div>.

$\odot$ Main content

The content of the page. It's what between the tag <div id="content">...</div>

$\odot$ Content table

This shows a quick link to each section of the page. It's the tag <div id="anchors"></div>. You don't have to edit it by hand. The script automatically update it.
Sections
You can add numbered sections with the tag "section" 

    <section label="anchor_name">Section name</section>
This will appear in the content table automatically. You can modify this in the "Section symbols" section of script.js
Citations
You can give a linked source with the tag "cite": 

    <cite src="source.html"></cite>
This is what it looks like. You can use the usual hyperlink tag, but I already modify it so it will pop up a new tab. I also make its appearance unified with a simple tag. You can modify its appearance in the "Citation" section of script.js
Wrappers
Sometimes, an object can be too large for the webpage. You can put your object inside the tag "wrap": 

    <wrap>
        //*your object here*//
    </wrap>

Equations
Just like LaTeX, you can use the dollars symbol to display equations. Use \$ for in-line equations and \$\$ for DisplayMath mode. Note that you can also use \$\$ for aligned equations (I modified it myself). Code 

    This is an in-line equation $E=mc^2$.
    This is a DisplayMath mode equation:
    \$\$
    x=\frac{-b\pm\sqrt{b^2-4ac}}{c^2}.
    \$\$
    And this is the aligned equations:
    \$\$
    f&=\frac{d}{dx}x^2\\
    &=2x.
    \$\$

Result This is an in-line equation $E=mc^2$. This is a DisplayMath mode equation: $$ x=\frac{-b\pm\sqrt{b^2-4ac}}{c^2}. $$ And this is the aligned equations: $$ f&=\frac{d}{dx}x^2\\ &=2x. $$ All of the DisplayMath equations are already inside the wrapper tag: Code 

    \$\$
    \text{This is a very long equation:}
    ~\int d^3x\left\{x+x^2+x^3+\sin(x)+2\sin(x)\cos(y)\right\}
    + \prod_i k_i\times\left[\exp(2i\pi y)+\cos(\tanh(5x))\right]
    \$\$

Result $$ \text{This is a very long equation:} ~\int d^3x\left\{x+x^2+x^3+\sin(x)+2\sin(x)\cos(y)\right\} + \prod_i k_i\times\left[\exp(2i\pi y)+\cos(\tanh(5x))\right] $$
Text boxes
You can add a text box with the "box" tag. This is good for drawing the reader's attention. Code 

    <box>
    A cat in a box.
    </box>

Result A cat in a box. You can adjust the style of the text box in style.css
Footnotes
If you want to add a small footnote over some sentence, use "note": Code 

    Some text here.<br>
    <note value="This is the footnote.">Some text</note> More text here.

Result Some text here.
Some text More text here.
On mobile, the footnote can be seen if you tap on the text. The footnote disappears if you tap somewhere else. You can adjust the style of the footnote in style.css. You can replace the footnote symbol $\textbf{[?]}$ in the "footnote" part of the "Macros" section in script.js.
Quotes and code listing
You can list the code with the usual pre-code combo. Quoting normal text can be done with "quote": Code 

    <pre><code>
    Some code here.
    </code></pre>
    <br>
    <quote>
    Some text here.
    </quote>

Result 

    Some code here.

Some text here.
