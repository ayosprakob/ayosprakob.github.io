Armillary sphere builder

Step 1: Representation expansions

Add sets of irreps to be used in the character/series expansion. Since each term in the action can be expanded with different series, you can add custom sets here for each of them.



Notation: For an irrep $(n_1,n_2,...,n_{N-1})$, the number $n_i$ denotes the number of boxes in the $i$th row of a Young tableau (English notation).

Note: for fermionic terms, the set is usually {(0), (1)}.





Step 2: Wilson loops

This section counts the number of Wilson loops in your action.




Write an expression to describe your operator. Strictly use the following notation:
  • Write U_\mu to denote a link variable in the \mu direction.
    You can replace \mu by any Greek or Latin letters (e.g., \nu, \sigma, etc.—small letters only).
  • The inverse is given by U'_\mu.
  • Do not use brackets.
  • Polyakov loop is denoted by L_\mu, where \mu is the wrapping direction.
  • We currently do not support operators with non-local separation.
